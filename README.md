# Sistema de Orçamento de Aluguel

## Descrição

Este projeto foi desenvolvido em Python com o objetivo de realizar orçamentos de aluguel de imóveis.

O sistema permite calcular valores de aluguel para apartamentos, casas e estúdios, incluindo adicionais, descontos e parcelamento do contrato.

Além disso, o programa gera automaticamente um arquivo CSV contendo os valores mensais do contrato.

---

## Funcionalidades

- Escolha do tipo de imóvel
- Cálculo automático do aluguel
- Adicional por quartos
- Adicional por garagem
- Desconto para clientes sem filhos
- Parcelamento do contrato
- Geração de arquivo CSV
- Tratamento de erros e validações

---

## Tecnologias Utilizadas

- Python 3
- Biblioteca CSV
